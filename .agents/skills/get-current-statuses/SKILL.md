# Get Current Statuses

Call the script [get_current_statuses.py](../../scripts/get_current_statuses.py) to check the active git branches and working copy statuses of all kb packages inside the `remotes/` folder.

## Usage

```shell
uv run scripts/get_current_statuses.py
```

### Examples
```shell
uv run scripts/get_current_statuses.py
```
